Flex布局 布局功能基于angular/flex-out组件。

布局API

flex布局通过HTML元素的属性来控制布局方式。支持行列方式的布局方式，支持响应式布局.

属性 赋值 说明
fxLayout row|column|row-reverse|column-reverse 布局属性决定html元素内所有子元素的布局方式
fxLayoutAlign <main-axis> <cross-axis>
main-aixs: start |center | end | space-around | space-between
cross-axis: start | center | end | stretch 		对齐方式
fxLayoutGap % | px | vw | vh 布局元素间的间距

子元素通过HTML属性调整布局

行式布局

行式布局内所有元素在同一行内排列.
