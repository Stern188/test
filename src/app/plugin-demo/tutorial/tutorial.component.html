# NG Cloud web UI 框架 NG Cloud Web 框架采用 angular 4+ 为基础引入多种开源组件进行组合而成。 默认采用google material design样式。 支持bootstrap布局。 ## 特性 - 为控制台设计的管理界面框架 - 前后端完分离架构设计 - 基于angular 并随时同步更新最新版本支持，有良好的社区支持 - 支持组件统一样式 - 支持forum表单根据json配置生成，确保界面风格一致 - 支持mock API测试，前后端开发过程分离 - 支持rxjs 提供流式开发 - 界面组件、界面模块化，便于复用 ## 目前引入组件列表 | 组件 | Demo URL | 功能 | | ------------- |:-------------:| -----:| |
