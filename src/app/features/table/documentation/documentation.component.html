API

  • Table(<ngx-datatable>标签上的属性)
    • Inputs

      • columnMode

        The mode which the columns are distributed across the table. For example: flex will use flex-grow api, force will distribute proportionally and standard will just distrbute based on widths. Default value: standard

      • columns

        Array of columns to display.

        要显示的列数组。

      • count

        The total count of all rows. Default value: 0

        所有行的总数。

      • externalPaging

        Should the table use external paging vs client-side. Default value: false

        该table应该使用外部分页还是客户端吗，默认为false.

      • externalSorting

        Should the table use external sorting vs client-side. false

        该table应该使用外部排序还是客户端吗，默认为false.

      • footerHeight

        The height of the footer in pixels. Pass a falsey for no footer. Default value: 0

        页脚的高度,单位为像素。通过设置false来去掉页脚，默认为0.

      • headerHeight

        The height of the header in pixels. Pass a falsey for no header. Default value: 30

        页眉的高度,单位为像素。通过设置false来去掉页眉，默认为30.

      • limit

        The page size to be shown. Default value: undefined

        要显示的每页条数.默认为undefined.

      • offset

        The current offset ( page - 1 ) shown. Default value: 0

        当前偏移量（page- 1）显示.默认为0.

      • rowHeight: Function|number|undefined

        The height of the row.

        行高

        Pass undefined for fluid heights, this is not possible in virtual scrolling.

        If using virtual scrolling, you must pass a function or a number to caculate the heights.

        如果使用上下滚动，你必须传递一个函数或一个数字计算高度。

        Using a function, you can bind individual row heights such as:

        使用函数，可以绑定单个行高度，例如

                        getRowHeight(row){
                  // set default
                  if(!row) 
                  return 
                  50;
                
                  // return my height
                  return row.height; 
                }

      • rows

        Array of rows to display.

        要显示的行数组。

      • scrollbarH

        Enabled horizontal scrollbars.Default value: false

        启用水平滚动条。默认false，不启动

      • scrollbarV

        Enable vertical scrollbar for fixed height vs fluid. This is necessary for virtual scrolling.Default value: false

        使用固定高度来启用垂直滚动条。这对于上下滚动是必要的。默认false，不启动

      • displayCheck

        A function you can use to check whether you want to show the checkbox for a particular row based on a criteria. Example: (row, column, value) => { return row.name !== 'Ethel Price'; }

        可以使用一个函数来设置您希望显示的特定行的复选框，比如 (row, column, value) => { return row.name !== 'Ethel Price'; }。

      • selected

        List of row objects that should be represented as selected in the grid. It does object equality, for prop checking use the selectCheck function.

        Default value: []

        被选中的行对象列表数据。默认[]。

      • selectionType

        Type of row selection. Options are single, multi, multiClick and chkbox. For no selection pass a falsey. Default value: undefined

        行选择类型。参数有single、multi、multiClick和checkbox。如果没有选择则设置false。默认undefined。

      • selectAllRowsOnPage

        点击全选复选框时是否选中全部数据，默认true，选择本页数据。

      • sortType

        Single vs Multi sorting. When in single mode, any click after the initial click will replace the current selection with the next selection. In multi selection mode, any incremental click will add to the current selection array. Default value: single

      • rowClass

        A function that will invoked with each row's properties. The result of the function can be a string or object with key/boolean like: 

                        
                  getRowClass(row) {
                    return {
                      'is-age-10': row.age === 10
                    }
                  }

      • groupRowsBy

        按哪个字段进行分组。

      • groupExpansionDefault

        默认是否展开分组，默认为false，不展开。

    • Outputs

      • activate

        A cell or row was focused via keyboard or mouse click.

                        {
                  type: 'keydown'|'click'|'dblclick'
                  event
                  row
                  column
                  value
                  cellElement
                  rowElement
                }

        单元格或行的键盘或鼠标回调方法

      • page

        The table was paged either triggered by the pager or the body scroll.

                        {
                  count
                  pageSize
                  limit
                  offset
                }

        每当点击分页、排序或滑动滚动条时触发。

      • tableContextmenu

        The context menu was invoked on the table.

                        {
                  event,
                  type,
                  content
                  }

        table上面调用了上下文菜单。

      • select

        A cell or row was selected.

                        {
                  selected
                }

        选择单元格或行时的回调方法

  • Column(<ngx-datatable-column>标签上的属性)
    • Inputs

      • name: string

        Column label. If none specified, it will use the prop value and decamelize it.

        列标签。如果没有指定，将使用prop的值以及decamelize它。

      • prop: string

        The property to bind the row values to. If undefined, it will camelcase the name value.

        要绑定的属性列的值。如果是undefined，将会用name属性对应的值。

      • flexGrow: number

        The grow factor relative to other columns. Same as the flex-grow API. It will any available extra width and distribute it proportionally according to all columns' flexGrow values. Default value: 0

      • maxWidth: number

        Maximum width of the column in pixels. Default value: undefined.

        列的最大宽度，单位为像素，默认undefined。

      • minWidth: number

        Minimum width of the column in pixels. Default value: 100.

        列的最小宽度，单位为像素，默认100。

      • width: number

        The width of the column by default in pixels. Default value: 150.

        列的宽度，单位为像素，默认150。

      • resizeable: boolean

        The column can be resized manually by the user. Default value: true.

        列的大小可以由用户手动伸缩。

      • sortable: boolean

        Sorting of the row values by this column. Default value: true.

      • draggable: boolean

        The column can be dragged to re-order. Default value: true.

        拖拽列以重新排序。

      • canAutoResize: boolean

        Whether the column can automatically resize to fill extra space. Default value: true.

        列是否可以自动调整大小以填充额外的空间。

      • checkboxable: boolean

        Indicates whether the column should show a checkbox component for selection. Only applicable when the selection mode is checkbox.

        指示列是否应该显示复选框组件以供选择。只适用于选择模式为 checkbox。

      • headerCheckboxable: boolean

        Indicates whether the column should show a checkbox component in the header cell. Only applicable when the selection mode is checkbox.

        指示列是否应该在标题单元格中显示复选框组件。只适用于选择模式为 checkbox。

      • editable: boolean

      • frozenLeft: boolean

        Determines if the column is frozen to the left. Default value: false

        确定列是否冻结到左侧。默认false

  • Row Detail(<ngx-datatable-group-header>、<ngx-datatable-row-detail>标签上的属性)
    • Inputs

      • rowHeight: Number|Function

        Height of the detail row.

        detail行的高度

        Template 

                        
                  <ngx-datatable-row-detail [rowHeight]="getHeight" #myDetailRow (toggle)="onDetailToggle($event)">
                  <ng-template let-row="row" ngx-datatable-row-detail-template>
                  <div><strong>Address</strong></div>
                  <div>row.address.city}}, row.address.state}}</div>
                  </ng-template>
                  </ngx-datatable-row-detail>

        Typescript 

                        
                  getHeight(row: any, index: number): number {
                    return row.someHeight;
                  }
    • Outputs

      • toggle

        Row detail row was toggled.

        detail行的触发方法

                        {
                  type: 'all' || 'row'
                  value: boolean || row object
                }
